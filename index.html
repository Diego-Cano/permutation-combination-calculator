How it Works

This calculator determines the number of ways to arrange or choose items from a set:

  • Permutation: The number of different possible arrangements of items where order matters.
  • Combination: The number of different possible selections of items where order doesn't matter.

For example, with n=5 and r=3:

  • P(5,3) = 5! / (5-3)! = 5! / 2! = 120 / 2 = 60
  • C(5,3) = 5! / (3! × (5-3)!) = 5! / (3! × 2!) = 120 / (6 × 2) = 10
